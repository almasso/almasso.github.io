Netscape Features List

Serves as a drop-in replacement for NCSA Mosaic on Microsoft Windows, Macintosh, and X Window System computers.

Interoperates with all HTTP-based clients and servers.

Delivers improved performance via:
Multiple, simultaneous image loading. Parallel loading can increase speed by nearly 10 times.
Continuous document streaming, documents appear almost instantaneouly.
Document and image caching.
Native JPEG image decompression.
GUI:
A toolbar for frequently used commands.
A network transfer progress indicator.
Easy access to bookmarks and history, bookmark menu has cascading menus to represent bookmark heirachy
Provides a common look, feel, and behavior across platforms.
HTML:
Supports Inline images in GIF, XBM and JPEG format.
The addition of the JPEG format represents a significant improvement over GIF and XBM in the area of image quality (24 bit color) and size.
Support advanced HTML constructs such as image maps and forms.
Extra HTML tags added to enhance documents.
HTTP:
Uses a "smart" caching scheme to greatly increase the speed of commonly visited pages.
FTP:
Supports an enhanced and highly detailed listing of files and directoris on FTP servers.
Uses connection caching to greatly increase the speed of file browsing.
USENET News:
Threaded newsgroup listings. Allows subscribing and unsubscribing from newsgroups.
Keeps track of previously read articles and shows only unread articles when desired.
Supports news posting and mail replies.
Gopher:
Support for Gopher 1.0
Bookmarks:
Supports a completely heirachical grouping of bookmarks including headers and separators.
Allows easy editing and reordering of bookmark items.
Allows easy exporting and viewing of bookmarks in HTML format.
Ability to import bookmark items from any HTML page.
Allows easy searching of bookmark items.
